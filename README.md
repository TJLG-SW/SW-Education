# 🏥 社会工作伦理案例模拟器

> **SW-Education / 1_1**
>
> 一个面向社会工作专业学生与在职人员的 AI 驱动伦理决策实训工具，聚焦医务社工场景中的安宁疗护与伦理两难抉择。

---

## 📖 项目简介

本模拟器通过真实感的角色扮演对话，让学习者扮演医院社工"李社工"，在一个 11 岁癌症晚期患儿（小明）及其家庭的真实场景中，面对一系列伦理困境并作出决策。

模拟结束后，系统将基于学习者的对话表现自动生成个性化伦理评估报告，帮助反思与成长。

**适用对象**：社会工作本科生、研究生及在职社工  
**应用场景**：课堂实训、自主学习、学术研究数据采集

---

## 🎭 案例背景

> 患儿陈小明，11 岁，非霍奇金淋巴瘤晚期。医疗团队已停止化疗，预计存活期 1–3 个月。学习者将作为医务社工介入，与患儿母亲刘雪梅、父亲陈国强及患儿本人展开对话。

---

## 🗂️ 剧情节点

模拟器共设 **5 个递进式剧情节点**，每个节点考察不同的伦理维度：

| 节点 | 主题 | 核心伦理议题 |
|------|------|-------------|
| 节点 1 | 情绪崩溃 | 同理心应用 vs 防止过度共情、专业边界 |
| 节点 2 | 基金会请求 | 诚实告知 vs 满足案主需求、最小伤害原则 |
| 节点 3 | 小明的疑问 | 儿童知情权、渐进式告知、案主自决 |
| 节点 4 | 父亲沟通 | 家庭系统协调、沉默型案主的情感接触 |
| 节点 5 | 家庭决策 | 临终关怀方案的综合伦理决策 |

每个节点设有多选项，不同选择影响伦理得分并触发不同的后续剧情走向。

---

## 🧩 功能特性

- **AI 动态对话**：基于 DeepSeek API 实时生成 NPC 角色回应，对话自然真实
- **多维伦理评分**：涵盖伦理识别、诚实告知、案主自决、专业关系、系统协调 5 个维度
- **进度自动保存**：通过 `localStorage` 保存对话进度，支持中断后续玩（有效期 24 小时）
- **个性化报告**：模拟结束后 AI 自动生成含节点分析、成长建议、反思题的 Markdown 报告
- **PDF 导出**：支持将评估报告导出为 PDF 文件（依赖 jsPDF CDN）
- **数据采集**：集成 Airtable，将匿名化决策数据上传供学术研究使用
- **响应式布局**：支持桌面端与移动端访问

## 🚀 本地运行指南

### 前置要求

- **Python 3.x**（用于启动本地静态文件服务器）
- **Node.js**（可选，仅构建压缩版本时需要）
- **DeepSeek API Key**（必须，用于 AI 对话功能）
- 现代浏览器（Chrome / Firefox / Edge，推荐 Chrome）

---

### 第一步：获取 DeepSeek API Key

1. 前往 [DeepSeek 开放平台](https://platform.deepseek.com/) 注册账号
2. 在控制台创建 API Key
3. 妥善保存 Key（格式形如 `sk-xxxxxxxxxxxxxxxx`）

---

### 第二步：配置 API Key

**方式一：通过浏览器 localStorage 配置（推荐）**

启动服务后，在浏览器控制台（按 `F12` → Console）输入以下命令：

```javascript
localStorage.setItem('DEEPSEEK_API_KEY', 'sk-你的API密钥');
location.reload();
```

此方式 Key 仅存储在本地浏览器中，不会写入代码文件。

**方式二：直接写入代码（本地调试用）**

打开 `1_1/api.js`，找到文件顶部的配置对象，将 `apiKey` 填入：

```javascript
const DEEPSEEK_API_CONFIG = {
  apiKey: 'sk-你的API密钥',   // ← 填入此处
  baseUrl: 'https://api.deepseek.com/v1/chat/completions',
  model: 'deepseek-chat',
  timeout: 30000
};
```

> ⚠️ **安全提示**：请勿将含有真实 API Key 的代码提交至公开仓库。

---

### 第三步：启动本地服务器

进入 `1_1` 目录，运行 Python HTTP Server：

```bash
cd 1_1
python -m http.server 8080
```

然后在浏览器访问：

```
http://localhost:8080
```

> ⚠️ **注意**：必须通过 HTTP Server 访问，直接双击打开 `index.html`（`file://` 协议）会因浏览器安全策略导致 JS 文件加载失败。

---

### 第四步（可选）：构建压缩版本

如需部署压缩后的版本，需先安装 Node.js，然后执行：

```bash
cd 1_1
node build.js
```

构建产物将输出至 `1_1/dist/` 目录，启动方式：

```bash
cd 1_1/dist
python -m http.server 8080
```

## ⚙️ 常见问题

**Q：页面显示"脚本文件加载失败"**  
A：请确认是通过 `http://localhost:8080` 访问，而非直接打开本地文件。

**Q：对话没有 AI 回复，一直转圈**  
A：请检查 API Key 是否正确配置，以及网络是否能访问 `api.deepseek.com`。可按 `F12` 查看控制台报错信息。

**Q：API 请求超时**  
A：默认超时为 30 秒，网络不稳定时可能触发。稍后刷新页面重试，进度会自动恢复。

**Q：想清除已保存的进度重新开始**  
A：在浏览器控制台执行 `localStorage.removeItem('sw_sim_backup')` 后刷新页面即可。
